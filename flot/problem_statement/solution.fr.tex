            {Problème} Pour une arrivée donnée trouver à quelle rivière rivière il faut déposer le bâteau. Calculer ça pour toutes les arrivés.

        \pause
        \textbf{Solution}: Faire un multisource 0-5 BFS $\mathcal{O}(n + m)$
